下面这套就按你截图里的页面来调。先把旧方案另存一个预设，新的从零开始录。

**1. 合频页面**
目标：蓝牙直连 DSP，只用蓝牙输入，原车高电平/低电平全部不要混进来。

| 通道 | 高电平1-6 | 低电平左/右 | 蓝牙左 | 蓝牙右 |
|---|---:|---:|---:|---:|
| CH1 前左高音 | 全 0 | 全 0 | 100 | 0 |
| CH2 前右高音 | 全 0 | 全 0 | 0 | 100 |
| CH3 前左中低音 | 全 0 | 全 0 | 100 | 0 |
| CH4 前右中低音 | 全 0 | 全 0 | 0 | 100 |
| CH5 后左 212C | 全 0 | 全 0 | 100 | 0 |
| CH6 后右 212C | 全 0 | 全 0 | 0 | 100 |
| CH7/CH8 | 全 0 | 全 0 | 0 | 0 |

截图里你有些通道同时混了低电平和蓝牙，这个一定改掉。

**2. 通道页面**
按每个 CH 点进去改：

| 通道 | 通道类型 | 增益 | 相位 | 高通 | 低通 |
|---|---|---:|---|---|---|
| CH1 | 前左高频 | -4dB | 正相 | 巴特沃斯 3600Hz / -24dB | 关闭 |
| CH2 | 前右高频 | -4dB | 正相 | 巴特沃斯 3600Hz / -24dB | 关闭 |
| CH3 | 前左中低频 | 0dB | 正相 | 巴特沃斯 63Hz / -24dB | 巴特沃斯 3600Hz / -24dB |
| CH4 | 前右中低频 | 0dB | 正相 | 巴特沃斯 63Hz / -24dB | 巴特沃斯 3600Hz / -24dB |
| CH5 | 后左全频 | -15dB | 正相 | 巴特沃斯 100Hz / -24dB | 巴特沃斯 8000Hz / -12dB |
| CH6 | 后右全频 | -15dB | 正相 | 巴特沃斯 100Hz / -24dB | 巴特沃斯 8000Hz / -12dB |

如果软件增益不好调负数，就把 CH1/2 设 0，CH3/4 也 0，然后用 Mosconi 的高音通道增益物理旋钮压低一点。后声场必须明显小，听起来只增加空间，不能听见后面在唱。

**3. 延时页面**
切到“毫秒 ms”，直接填：

| 通道 | 延时 |
|---|---:|
| CH1 前左高音 | 1.60ms |
| CH2 前右高音 | 0.10ms |
| CH3 前左中低音 | 1.35ms |
| CH4 前右中低音 | 0.00ms |
| CH5 后左 | 8.50ms |
| CH6 后右 | 7.80ms |

调完听单声道人声：人声偏左，就 CH1/CH3 各加 0.05-0.10ms；人声偏右，就 CH1/CH3 各减 0.05-0.10ms。

**4. EQ 页面**
先对 CH1-CH6 都点“重置均衡”。然后只填下面这些，其余波段增益保持 0。

CH1/CH2 前高音：

| 频率 | 增益 | Q值 |
|---:|---:|---:|
| 4000Hz | -1.0dB | 1.0 |
| 6300Hz | -1.0dB | 1.0 |
| 8000Hz | -1.5dB | 1.6 |
| 10000Hz | -1.0dB | 2.0 |

CH3/CH4 前中低音：

| 频率 | 增益 | Q值 |
|---:|---:|---:|
| 200Hz | -1.5dB | 1.2 |
| 250Hz | -1.0dB | 1.3 |
| 630Hz | -1.0dB | 1.0 |
| 2500Hz | -1.0dB | 1.5 |

CH5/CH6 后声场：

| 频率 | 增益 | Q值 |
|---:|---:|---:|
| 250Hz | -2.0dB | 1.0 |
| 2500Hz | -3.0dB | 1.0 |
| 6300Hz | -3.0dB | 1.0 |

**调音顺序**
先把 CH5/CH6 后声场静音，只听 CH1-CH4，把人声调到前挡风玻璃中间。前声场稳定后，再打开后声场，后声场音量从 -18dB 慢慢加，最多到 -12dB 左右，听到空间变大但人声不往后跑就停。

高音分频不要低于 3200Hz；你这套我建议固定 3600Hz / 24dB，安全、耐听，也更适合日常大音量。
