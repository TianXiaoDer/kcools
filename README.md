# 一、先确认系统定位

你的喇叭分布是：

| 单元     | 位置       | 声场作用         |
| ------ | -------- | ------------ |
| 高音 ×2  | 前挡玻璃左右下角 | 空气感、细节、声场高度  |
| 中音 ×2  | 主驾/副驾门旁  | 人声主体、结像、乐器定位 |
| 中低音 ×2 | 后排座椅左右两侧 | 低频、鼓点、厚度     |

从专业调音角度看，这套结构的重点是：

> **前面负责人声和声场，后面的中低音只负责补低频，不能让人听出声音来自后排。**

所以这套车不能把后排中低音放得太高、太响、太宽频。否则会出现：

* 人声胸腔感跑到后面；
* 鼓点从后排打出来；
* 声场塌到座椅附近；
* 前挡玻璃声场起不来。

因此这套调音的核心是：

> **中低音低切高切收窄、音量压住、延时对齐，让低频“听起来在前面”。**

# 六、通道增益：不要再用原来的 +6、+3 思路

你原来的截图里有些通道 +6、有些 +3、有些 0。现在蓝牙直入后，建议全部重新做。

先这样起步：

| 通道       | 单元   | 初始增益 |
| -------- | ---- | ---: |
| CH1 左高音  | 左高音  | -4dB |
| CH2 右高音  | 右高音  | -2dB |
| CH3 左中音  | 左中音  | -3dB |
| CH4 右中音  | 右中音  |  0dB |
| CH5 左中低音 | 左中低音 | -3dB |
| CH6 右中低音 | 右中低音 | -2dB |

为什么左边普遍低一点？

因为主驾在左边，左侧喇叭离耳朵近，天然更响。
如果左右增益完全一样，通常人声会偏左。

后期调音时，主要看人声中心：

* 人声偏左：继续降低 CH1 / CH3；
* 人声偏右：提高 CH1 / CH3，或降低 CH2 / CH4；
* 低频在后排：降低 CH5 / CH6，或把低通从 250Hz 降到 200Hz；
* 高频刺耳：降低 CH1 / CH2，先不要急着动 EQ。

# 七、相位设置

先全部设为：

> **正相**

也就是：

| 通道  | 相位 |
| --- | -- |
| CH1 | 正相 |
| CH2 | 正相 |
| CH3 | 正相 |
| CH4 | 正相 |
| CH5 | 正相 |

然后再做两步相位验证。

## 1. 中低音和中音的相位

播放男声、人声鼓点、低频较稳的歌曲。

先听 CH3/CH4 + CH5/CH6。

然后把 **CH5、CH6 同时反相** 试一下。

比较两种状态：

* 哪个状态低频更结实；
* 哪个状态男声胸腔更自然；
* 哪个状态鼓点不空、不散；
* 哪个状态声音更靠前。

如果反相后低频更集中、更有力，可以保留 CH5/CH6 反相。
但不要只反一个，要么两个中低音一起正相，要么一起反相。

## 2. 高音和中音的相位

播放女声、钢琴、吉他、人声靠前的歌曲。

把 CH1/CH2 高音一起反相试一下。

比较：

* 人声是否更聚焦；
* 齿音是否更自然；
* 声场高度是否更稳定；
* 中高频有没有“空掉”。

一般起步建议正相，但最终以实际听感和测试为准。

# 八、延时：这是你原方案最大的问题

你原来截图里的延时几乎等于没做。
现在这套要重新做，尤其你中低音在后排，延时非常关键。

## 正确算法

以主驾耳朵为听音点，测量耳朵到每个喇叭的距离。

公式：

> **延时 =（最远喇叭距离 - 当前喇叭距离） ÷ 34.3**

单位：

* 距离用 cm；
* 算出来是 ms。

---

## 临时可用的主驾延时参考

在你没给我实际距离前，可以先用这套“起步值”：

| 通道  | 喇叭   |  临时延时 |
| --- | ---- | ----: |
| CH1 | 左高音  | 2.3ms |
| CH2 | 右高音  | 0.9ms |
| CH3 | 左中音  | 2.6ms |
| CH4 | 右中音  | 0.8ms |
| CH5 | 左中低音 | 1.3ms |
| CH6 | 右中低音 |   0ms |

这个假设是：右后中低音大概率是离主驾最远的单元，所以 CH6 作为 0ms 参考。

但这只是起步值，不是最终值。最终要用你实际测量的距离来算。

## 蓝牙直入的基础 EQ 思路

不要一开始就大幅拉 EQ。专业调音顺序是：

> 分频 → 相位 → 延时 → 电平 → EQ

EQ 是最后修饰，不是用来救系统结构问题的。

---

## 初始目标曲线

车内不建议调成完全平直。完全平直会显得薄、冷、刺。

建议目标是：

* 60–100Hz 稍微丰满；
* 200–500Hz 不浑；
* 1kHz 人声自然；
* 2k–4k 不顶耳；
* 6k–10k 不刺；
* 12k 以上保留空气感。

也就是整体大概是：

> **低频略高，中频干净，高频柔和延伸。**

---

## 可用的基础 EQ 建议

### CH5 / CH6 中低音

先少量处理：

| 频率    | 建议              |
| ----- | --------------- |
| 80Hz  | +1 到 +2dB，Q 0.8 |
| 125Hz | 0 或 -1dB        |
| 200Hz | -1 到 -2dB       |
| 250Hz | 如果低频从后面出来，-2dB  |

后排中低音最怕 160–250Hz 太多。
这个区域一多，声音就会往后跑。

---

### CH3 / CH4 中音

| 频率      | 建议               |
| ------- | ---------------- |

| 500Hz   | 如果声音闷，-1dB       |
| 1kHz    | 0                |
| 2kHz    | 0 或 -1dB         |
| 3.15kHz | 如果人声顶耳，-1 到 -2dB |

中音不要乱挖。
中音一乱，人声就散。

---

### CH1 / CH2 高音

| 频率      | 建议              |
| ------- | --------------- |

| 6.3kHz  | 如果亮，-1dB        |
| 8kHz    | 如果齿音重，-1 到 -2dB |
| 10kHz   | 0 或 -1dB        |

你的高音在前挡玻璃下角，反射会明显，所以高频宁可柔一点，不要一开始拉得太亮。
